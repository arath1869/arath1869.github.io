My Work

Instafake is an Instagram clone, it starts off on a Login page, Signup is possible with passwords being fully encrypted when stored in the database (SQLAlchemy), the back-end utilizes Python, Flask, and the entire application is containerized in a docker container. Front-end technologies include React.js / Redux.js. Front-end makes calls to the back-end API to create, render, update and delete data (CRUD). When a user is logged in, they can like or comment on their friends' recent posts, they can also search for friends as well as follow/unfollow them as they please.
When users post an image, this gets stored on Amazon Web Services (AWS), allowing for a full and dynamic user experience. The entirety of the website utilizes cutting edge algorithms and programming best practices to ensure a page refresh is never needed, thus, improving speed, reliability and ultimately, user experience.
Multiple modals are also utilized throughout the website to allow the user to access the website's full potential without having to navigate away from a page they were on.

DrinkReviews is a website, loosely based on Yelp. The back-end utilizes Node.js, Express.js and Sequelize/PostgreSQL, and the front-end utilizes React.js/Redux.js and makes calls to the back-end API to create, render, update and delete data(CRUD).
The website may be utilized by any user, and you do not have to be logged in to view reviews/ search for drinks. However, you do have to be logged in, to post any drink or review.
When non-logged in users attempt to post a new drink or review, a login modal appears, making them log in, or signed up before posting.
When a user posts a drink, they upload a photo of the drink, this gets stored on Amazon Web Services (AWS), providing our user a seamless experience. Thanks to React/Redux, as well as the most cutting edge algorithms and programming best practices, the entirety of the website can be utilized to it's full potential without ever needing a page refresh. This ensures speed, reliability and ultimately a great user experience.
